
# IT Service Booking System

A web-based platform for companies to showcase their services and for clients to request customized services. Clients can choose services, add extra features, provide project details, and submit requests directly to the company. Admins can manage projects, track revenue, and view analytics.


## Features

### Client Side
- **Service Browsing:** View all services offered by the company.  
- **Service Customization:** Add extra features or requirements for each service.  
- **Project Details Submission:** Provide project description, expected timeline, and budget.  
- **Request Submission:** Send service requests directly to the company for approval.  
- **Client Dashboard:** Track status of requests (Pending → In Progress → Completed) and previous submissions.

### Admin / Service Provider Side
- **Project Management:** View all client requests with details like project type, features requested, budget, and timeline.  
- **Approval Workflow:** Approve or reject client project requests.  
- **Dashboard Analytics:**  
  - Total projects (Pending, Completed, Rejected)  
  - Revenue generated (completed projects)  
  - Pipeline value (projects in progress or pending)  
  - Service-wise performance reports  
- **Graphs & Reports:** Visual representations of:  
  - Number of projects by status  
  - Popular services and add-on features  
- **Client Insights:** See client history, repeated requests, and feedback.  

---

## Technologies Used
- **Frontend:** React.js, TypeScript  
- **Styling:** Tailwind CSS  
- **Charts & Analytics:** Chart.js / Recharts  
- **Version Control:** Git & GitHub
